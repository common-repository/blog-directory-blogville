Blogville Blog Direcotry Plugin updates your blog listing at the blogville blog directory with the date your blog was last updated.

== Description ==

This plugin will send a notice to the blogville blog directory whenever you add a new post or a new page to your blog.
This will help keep your listing updated and will allow it to stand out to better promote your blog.

== Installation ==

Simply copy the file blogville-plugin.php to your plugins directory.
Login to your blog and activate the plugin.
For the plugin to work you need to have an approved blog on the blogville blog directory (blogville.us).
